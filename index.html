Оценка продуктового запуска

Критерий 0 баллов 1 балл 2 балла Комментарий
1. Reach (Охват)
Узкая ниша / выделенный сегмент
  • Клиенты с ограниченной потребностью
  • Данные об охвате отсутствуют
Крупный сегмент
  • Пользователи определенной услуги
  • Определенный регион
Большая часть базы
2. Impact (Влияние)
Косвенное влияние
  • Низкий эффект
  • Эффект не поддается измерению
Заметное влияние
  • Явное улучшение usability
  • Упрощение процесса / Эффект измерим
Ощутимый инкремент
  • Влияние на ключевые показатели (выручка, АБПП, снижение оттока)
3. Validation (Обоснование)
Явные обоснования отсутствуют
  • Тестируется бизнес-гипотеза
  • Провалидированные результаты отсутствуют
Есть значимые обоснования
  • Бизнес-кейс / Опросы / Аналитика
  • Подтвержденная необходимость
Подтверждена высокая важность
  • Регулятор / Топ-менеджмент
  • Стратегическое направление
4. СХ-сложность и степень изменений
Линейный путь
  • 1-2 шага / Минорные правки
  • Изменений для клиента нет
Разветвленные пути
  • Новые шаги в сценарии
  • Развилки и вариативность
Комплексные или новые пути
  • Новые сценарии
  • Кардинальная смена логики
5. Токсичность
Не выявлена
Умеренная
  • Баннеры, уведомления, "пуши"
  • Активное продвижение новых условий
Высокая
  • Dark-паттерны / Повышение цены
  • Монетизация ранее бесплатного
6. Реализация
Стандартное решение
  • Повтор существующих механик
  • По аналогии с другими продуктами
Требуются нововведения
  • Адаптация существующих решений
Нестандартное решение
  • Создание уникального взаимодействия
7. Влияние на обращаемость
Не прогнозируется
  • Изменения бесшовные
  • Коммуникации не задействованы
Умеренное влияние
  • Небольшой охват
  • Задействован 1 вид коммуникаций
Риск повышенной обращаемости
  • Большой охват
  • Несколько каналов связи
